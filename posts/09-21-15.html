Today I Made Hipster Feed
Monday, September 21, 2015

After checking out sferik's fantastic command-line power tool for Twitter, I wanted to contribute to it by adding image functionality to it ala Twitter's official timeline. I envisioned using ascii art to display the images inline with the tweets. I did a bit of Googling and came across Ionica Bizau's image-to-ascii converter. Wanted to try it out, I decided to make an Instagram client for ANSI consoles to dip my toes in the ascii-art water. After a few hours of work, Hipster Feed was born.


The user interface non-existant, and I haven't even REPL-ized it -- but don't judge! This is my first commandline tool, and my first time using node.js to make anything. Currently, to run the app, you need to pass in an Instagram user acces_token, so that Hipster Feed can include it in it's API calls. This is ugly. That being said, it works, and it was a fun little experiment to do. I'm probably going to work on it a little more and add more commands for users to be able to interact with their accounts better and more comprehensively. For now though, it just a fun little experiment, and a great proof of concept that ascii-rendered images can be used to create richer user experiences for those (few) seeking to access their social media accounts in their console. Code is included below.



function renderFeed() {
  var options = {
    path : "",
    pxWidth: 6,
    aRatio : false
    https.get('https://api.instagram.com/v1/users/self/feed?access_token=' + access_token, function(res) {
      var body = '';

      res.on('data', function(chunk) {
        body += chunk;
        res.on('end', function() {
          var response = JSON.parse(body);
          for (var i = 0; i < response["data"].length; i++) {
            options.path = response["data"][i]["images"]["standard_resolution"]["url"];
            ImageToAscii(options, function(err, converted) {
              console.log(err || converted);
            });
          }
        }).on('error', function(e) {
          console.error(e);
        });
      }
    }
  }
}
